# 使用工作台新建插件

这个文档描述了如何使用工作台新建插件。

  1. 在工作台的“创作”分页，点击新建基岩版服务器


![image-20220411163053963](/dev/mcmanual/mc-dev/assets/img/image-20220411163053963.d83fff98.png)

  2. 点击空白插件的新建按钮，以打开新建插件的弹窗。


![image-20220411162929246](/dev/mcmanual/mc-dev/assets/img/image-20220411162929246.8c03ae43.png)

  3. 空白插件的说明是新建插件的简单介绍，下面的详细说明是插件的B站视频教程的链接。


![image-20220411163207919](/dev/mcmanual/mc-dev/assets/img/image-20220411163207919.cd93e72e.png)

  4. 在基岩版服务器 - 插件分页里也有新建插件按钮，点击这个也可以弹出弹窗。


![image-20220411163546702](/dev/mcmanual/mc-dev/assets/img/image-20220411163546702.912ec8a2.png)

  5. 新建插件的弹窗如下，输入团队名称，插件名称等，并进行勾选，点击创建即可完成一个空白插件的创建。


![image-20220411163641320](/dev/mcmanual/mc-dev/assets/img/image-20220411163641320.5192fb43.png)

  6. 创建完成的插件会直接显示在工作台中。


![image-20220411163922869](/dev/mcmanual/mc-dev/assets/img/image-20220411163922869.6a84f11b.png)
