# 3.1

2024.10.17：版本号（V3.1 BE1.20.12）

包括Mod PC包，手机测试版启动器，和服务器引擎。

示例Demo：[前往此处下载 (opens new window)](<https://mc.163.com/dev/mcmanual/mc-dev/mcguide/20-%E7%8E%A9%E6%B3%95%E5%BC%80%E5%8F%91/13-%E6%A8%A1%E7%BB%84SDK%E7%BC%96%E7%A8%8B/60-Demo%E7%A4%BA%E4%BE%8B.html>)

## 一、更新时间表

  1. 在2024年10月17日上线3.1第一个Beta版。
  2. 在2024年10月31日上线3.1第二个Beta版。
  3. 在2024年11月14日上线3.1稳定版。
  4. 在2024年11月22日，全渠道将更新3.1版本玩家包体，玩家将陆续更新到3.1版本，请开发者合理安排更新节奏。


## 二、API更新

  * 新增


  1. 新增[HideCrossHairGUI](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/原生UI.html#hidecrosshairgui>)(客户端)， 隐藏hud界面的十字准心显示

  2. 新增[ResetToDefaultValue](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/实体/属性.html#resettodefaultvalue>)(服务端)， 新增重置实体引擎属性到默认值接口

  3. 新增[ResetToMaxValue](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/实体/属性.html#resettomaxvalue>)(服务端)， 新增重置实体引擎属性到最大值接口

  4. 新增[EvalMolangExpression](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/实体/molang.html#evalmolangexpression>)(服务端)， 在实体上下文上执行molang表达式

  5. 新增[GetItemTexture](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/物品.html#getitemtexture>)(客户端)， 获取物品的贴图

  6. 新增[EvalMolangExpression](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/实体/molang.html#evalmolangexpression>)(客户端)， 在实体上下文上执行molang表达式

  7. 新增[PlayerPickupArrowServerEvent](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/事件/物品.html#playerpickuparrowserverevent>)(服务端)， 玩家即将捡起箭矢时触发


  * 调整


  1. 调整[SetAttrValue](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/实体/属性.html#setattrvalue>)(服务端)， 新增setDefault参数

  2. 调整[SetAttrMaxValue](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/实体/属性.html#setattrmaxvalue>)(服务端)， 补充说明SetAttrMaxValue会同时修改默认值

  3. 调整[GetItemBasicInfo](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/物品.html#getitembasicinfo>)(服务端)， 新增customTips字段，获取自定义物品的netease:customtips/自定义方块的netease:custom_tips中的配置

  4. 调整[GetItemBasicInfo](</dev/mcmanual/mc-dev/mcdocs/1-ModAPI/接口/物品.html#getitembasicinfo>)(客户端)， 新增customTips字段，获取自定义物品的netease:customtips/自定义方块的netease:custom_tips中的配置
