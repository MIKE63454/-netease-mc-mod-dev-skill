## 制作方法

  1. 在材质包目录创建名为 **blocks.json** 的文件。

![img](/dev/mcmanual/mc-dev/assets/img/3_1.32e1ea97.png)

  2. 将下方代码复制到 **blocks.json** 内。
[code] {
             "format_version":[
                 1,
                 1,
                 0
             ],
             "torch":{
                 "textures":"torch_on",
                 "blockshape":"torch",
                 "carried_textures":"missing_tile"
             }
         }
```
![img](/dev/mcmanual/mc-dev/assets/img/3_2.54aac261.png)

  3. 制作一个火把材质贴图，并保存到目录/textures/blocks，命名为missing_tile。

![img](/dev/mcmanual/mc-dev/assets/img/3_3.51f23b92.png)


完。
